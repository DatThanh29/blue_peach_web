# blue_peach_web
Hệ thống quản lý và bán hàng đa nền tảng cho cửa hàng trang sức bạc Blue Peach
## Blue Peach Jewelry E-commerce System

### Tech Stack
- Frontend: Next.js (App Router), TypeScript
- Backend: Node.js, Express, TypeScript
- Database: Supabase (PostgreSQL)
- Auth: Supabase Auth
- Storage: Supabase Storage (planned)

### Features (MVP)
- Product listing & search
- Product detail
- Cart (localStorage)
- Checkout & order creation
- Inventory update

### Project Structure
- frontend/: Next.js app
- backend/: Express API
